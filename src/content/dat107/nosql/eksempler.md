# Øvinger Og Eksempler

## Viktigste øvingsfiler

- `videoer/video-04-xpath/ovinger-med-løsningsforslag.md`
- `videoer/video-05-xpath-og-xslt/øving/løsningsforslag/studenter.xsl`
- `videoer/video-06-xml-postgresql/øving/studenter-xml.sql`
- `videoer/video-07-json-og-postgresql/øving/studenter-json.sql`
- `forelesninger/forelesning-01/02-birken/birken-paamelding.xsd`
- `forelesninger/forelesning-03/json/birken-paamelding.json`

## Dette må du kunne

- vite hvilke filer som trener hva
- bruke øvingsfilene som aktiv repetisjon, ikke bare lesestoff
- lese kritisk når eksempeldata ser bevisst feilaktige eller mistenkelige ut

## Hvorfor disse filene er viktige

| Fil | Trener |
| --- | --- |
| `ovinger-med-løsningsforslag.md` | XPath-drill |
| `studenter.xsl` | XSLT og beregninger |
| `studenter-xml.sql` | XML i PostgreSQL |
| `studenter-json.sql` | JSONB i PostgreSQL |
| `birken-paamelding.xsd` | XSD, regler og referanser |
| `birken-paamelding.json` | realistisk JSON med nested struktur |

## Vanlige feil

- bruke øvingsfilene passivt i stedet for å skrive egne uttrykk først
- stole blindt på eksempeldata uten å vurdere om noe er bevisst lagt inn som øvingsfeil

## Typiske eksamensoppgaver

- små XPath-uttrykk
- forslag til JSON- eller XML-struktur
- drøfting av dokumentvalg versus relasjonelt valg

## Hva du bør øve på

- velge én fil og lage tre egne spørsmål til den
- forklare hva filen trener og hvorfor den er relevant til eksamen
